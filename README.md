# Calculadora de Rectángulos

Programa simple en Python que calcula el área y el perímetro de un rectángulo a partir del ancho y alto ingresados por el usuario. También identifica si la figura es un cuadrado.

## Cómo usar
1. Ejecuta `rectangulo.py`
2. Ingresa el ancho y el alto cuando el programa lo pida
3. Lee los resultados en pantalla

## Lo que se aplica
- Tipos de datos: float, string y boolean
- Identificadores en snake_case
- Uso de funciones y condicionales

## Requisitos
Python 3.x
